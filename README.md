# 🏦 ChekScan : Système Intelligent de Numérisation de Chèques

**ChekScan** est une plateforme moderne permettant d'automatiser la lecture et la gestion des chèques bancaires. Grâce à une intelligence artificielle avancée, l'application extrait instantanément les informations critiques pour simplifier vos processus financiers.

## ✨ Fonctionnalités
- 📸 **Numérisation Automatisée** : Analyse instantanée des images de chèques (JPG, PNG).
- 🎯 **Extraction de Données** : Identification automatique de la banque, du montant, de la date et de la ligne magnétique (MICR).
- 💾 **Historique des Scans** : Sauvegarde sécurisée de chaque opération pour une consultation ultérieure.
- 🌓 **Interface Personnalisable** : Choix entre Mode Clair et Mode Sombre pour un confort optimal.
- 🌍 **Multi-Langues** : Interface disponible en Français et en Arabe.
- 🤖 **Assistant Intelligent** : Un agent conversationnel dédié pour vous accompagner dans l'utilisation.

---

## 🚀 Installation Rapide (Local)

L'application est entièrement conteneurisée pour garantir un fonctionnement immédiat.

1. **Lancement** :
   ```bash
   docker-compose up -d --build
   ```
2. **Accès** : Ouvrez **[http://localhost:8080](http://localhost:8080)** dans votre navigateur.

## 🏛️ Architecture Logicielle

Conçue pour la performance et la sécurité, l'application utilise une architecture robuste :
- **Backend** : Cœur de traitement intelligent basé sur Python.
- **Frontend** : Interface utilisateur réactive et élégante (Premium Glassmorphism).
- **Base de Données** : Persistance locale des données pour une confidentialité totale.
